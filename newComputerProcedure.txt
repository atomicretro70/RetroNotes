     b.  .vimrc, .bashrc
     c.  figletRandomFont.py, randomMessage.py
     d.  swdev directory

     a.  git clone https://...

26.  Configure vim
     a.  mkdir ~/.vim
     b.  place configForCode.vim into ~/.vim
     c.  place configForProse.vim into ~/.vim
     d.  place configForSlides.vim into ~/.vim
     e.  place my .vimrc in ~

     f.  install vim-plug

     f.  install airline
     f.  install powerline fonts

     g.  mkdir ~/.vim/thesaurus/
     h.  download both thesaurus from
         https://www.gutenberg.org/files/3202/files/

     i.  place in thesaurus directory

     j.  Might install gvim - not necessary though
     k.  May need to install NERDTree & AirLine

27.  tkinter
     a.  sudo apt-get install python3-tk
     b.  reboot linux
     c.  sudo apt-get install python3-setuptools
     d.  sudo easy_install suds
     e.  sudo apt-get install python3-matplotlib

28.  make command py
     a.  cd /usr/bin
     b.  sudo ln -s python3.8 py
     c.  Must remember to update this every time a new version is installed.
     d.  could instead make an alias in .bashrc the difference being that
     e.  the alias will only work for one user.

29.  update
     a.  sudo apt update
     b.  sudo apt upgrade

30.  Install the following commands using: sudo apt-get install <command>

     Useful Items
     a.  ubuntu-restricted-extras    restricted video codecs
     b.  neofetch  displays system info
     c.  htop      enhanced top command
     d.  tree      display the filesystem in a tree

     Fun Display
     e.  figlet    figlet (don't forget fonts into /usr/share/figlet)
         showfigfonts
     f.  Download and move as sudo extra figlet fonts /usr/share/figlet 
     g.  toilet    figlet clone but does some other stuff too like textboxes
     h.  lolcat    lolcat
     i.  cmatrix   display The Matrix movie screen

     Optional
     i.  nvim      neovim
     j.  gvim      gvim
     k.  nvim-qt   neovim-qt
     l.  screenkey show keypresses on the screen for doing videos.
